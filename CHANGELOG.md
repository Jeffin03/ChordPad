## [1.0.0] - 2026-04-18

### Added
- 7 chord slots mapped to scale degrees I through VII
- Tap to play staccato, hold to sustain per slot
- Two-hand piano voicing engine — right hand mid-high, left hand bass
- Full music theory engine — notes, scales, chords, variations, voicing
- Major and minor scale support
- Chord variations per slot — triad, 7th, maj7, m7, sus2, sus4, inv1, inv2
- Slash chord support with diatonic bass note picker
- Global key selector — root note and scale type
- Semitone transpose — shifts all voicings proportionally
- Velocity modes — Soft, Medium, Hard, Fixed
- Edit mode — toggle between play and edit without losing hold gesture
- SlotEditor bottom sheet with VariationPicker, VoicingPreview, and MiniPiano
- MiniPiano — two octave read-only keyboard highlighting active voicing notes
- Song presets — save, load, delete, export and import as JSON
- Preset bar with scrollable chips, long press to edit
- Zustand store with localStorage persistence
- Salamander Grand Piano samples via gleitz midi-js-soundfonts
- Tone.js sampler with interpolation across 24 anchor MP3 files
- PWA — installable on Android and iOS, works offline
- Deployed to GitHub Pages at jeffin03.github.io/ChordPad
